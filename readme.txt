## Development
== Description==
Plugin này tạo ra một thanh liên hệ nằm ngang trên giao diện mobile , plugin tương thích với tất cả các theme thịnh hành.

### Requirements
Tương thích với hầu hết các phiên bản wordpress và theme thịnh hành.
#### Features:

### Installation
- Cài đặt chat qua zalo, facebook , hotline
-Hỗ trợ tốt nhất PHP 7.0
- Tương thích với WordPress 5.0 trở lên

### Linting

== Installation ==

Cài đặt và tùy chỉnh như các bước dưới đây:
1. Tải \"Thanh-lien-he-mobile\" lên thư mục \"/wp-content/plugins/\".
2. Kích hoạt \"Plugins\" trong menu WordPress.
3. Tùy chỉnh thêm số thông tin vào và sử dụng.
